# Token Transfer Example (ERC20)

Steps:
1. Run `yarn chain` and `yarn deploy` (root or hardhat package) to deploy `TokenDemo`.
2. Open `/examples/token-transfer` and connect your wallet.
3. Click “Mint 1000 DEMO to me” to get tokens.
4. Enter a recipient and amount, then click “Send”.
5. Inspect events and balances in the Debug and Block Explorer pages.
