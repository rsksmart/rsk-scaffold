# NFT Minting Example (ERC721)

Steps:
1. Deploy with `yarn deploy` to create `NftDemo` with a default base URI.
2. Open `/examples/nft-minting` and connect your wallet.
3. Click “Mint NFT” to mint a token to your address.
4. Optionally call `setBaseTokenUri` to point to your own metadata.
5. Use Debug/Contracts to read tokenURIs and events.
